- External programs (fork, execvp, waitpid)
  Run non-builtin commands in a child: fork, execvp in the child, waitpid in the parent.
  Today only builtins run; unknown names do nothing.

- argv-style args for exec
  Split the line into a char *argv[] (program name + arguments) so execvp gets real args,
  not just “first word + rest of line” as one blob.

- history builtin (history.c empty)
  Record each submitted line and implement the history command to list (or recall) it,
  per the spec.

- input < and output >
  Open files and dup2 stdin/stdout before exec so cmd < infile and cmd > outfile work.

- one | pipeline
  Parse two commands around |, use pipe + dup2 so stdout of the left feeds stdin of the
  right (e.g. ls | wc -l).
